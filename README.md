# Lista de Exercícios 1 e 2  Programação em C

Sobre o projeto
Este repositório foi criado para armazenar:

os códigos em linguagem C
a organização das atividades da disciplina "Programação em C"
O objetivo é centralizar em um só lugar os materiais e as soluções desenvolvidas durante os estudos de Algoritmos e Lógica de Programação.

Este repositório ajuda no estudo de:

* lógica de programação
* variáveis e tipos de dados
* entrada e saída
* operadores matemáticos
* estruturas condicionais
* estruturas de repetição
* switch case
* organização de códigos em C
